3. 상속

extends 사용 -오버라이딩과 오버로디 
1.오버라이딩:
부모 클래스에게 상속받은 메소드를 자식 클래스에서 재정의하는것
리턴타입, 인수, 매개변수가 반드시 같아야 한다
2. 오버로딩:
하나의 클래스 내에서 이름이 같은 여러 개의 메소드를 정의하는것
매개변수의 타입이 다르거나 개수가 달라야한다
